# AGENTS.md

## プロジェクト概要
現場系個人事業主向けの作業履歴管理アプリを実装する。

## 開発方針
- Android先行
- ローカルファースト
- オフライン前提
- 3タップ以内操作
- 1秒以内記録

## 技術制約
- Flutter
- SQLite
- Riverpod
- Supabaseは後で追加

## MVP優先順位
1. クイック記録
2. 履歴一覧
3. 詳細編集
4. 未整理管理
5. マスタ管理
6. GPS
7. サジェスト
8. 地図

## 実装ルール
- 小さく実装する
- 常にビルド通す
- 過剰設計しない
- UIより導線優先

## テスト方針
- 実機なし前提
- Emulator + Unit Test
- Firebase Test Lab前提

## 完了条件
- ビルド通る
- 基本操作動く
- README更新されている
